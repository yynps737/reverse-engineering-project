平台介绍

高级逆向工程平台是一套集成的工具集，专为软件逆向分析、恶意软件分析和安全研究设计。本平台通过用户友好的Web界面，将多种专业逆向工具无缝集成，简化了复杂软件的分析过程。

主要功能包括静态分析、动态分析、脱壳、内存转储分析等，特别适用于分析受到VMProtect、Themida等商业壳保护的软件。

本平台仅供教育和研究目的使用，请在合法合规的前提下使用本工具。

安全研究工具

专为安全研究人员、逆向工程师和教育工作者设计

主要特性
静态分析

自动识别文件类型、检测壳和保护技术、提取特征信息

动态分析

运行时监控API调用、内存访问、网络通信和反调试行为

脱壳引擎

支持多种商业壳，包括UPX、VMProtect、Themida等

内存分析

内存转储、区段分析和IAT重建

网络监控

追踪和分析网络连接、HTTP请求和数据传输

详细报告

生成全面的分析报告，包括检测结果和建议

使用说明
快速开始：
  1. 在首页上传目标文件（支持EXE、DLL、SYS等可执行文件）
  2. 系统会进行初步分析，识别文件类型和架构
  3. 根据需要选择执行"静态分析"、"动态分析"或"脱壳"操作
  4. 等待分析完成，查看详细报告和结果文件
  5. 下载分析报告或处理后的文件进行后续研究
命令行使用：

除了Web界面，您还可以使用命令行直接调用核心工具：

python3 shell_detector.py 目标文件路径 - 执行静态壳检测
python3 dynamic_analyzer.py 目标文件路径 -t 超时时间 - 执行动态分析
python3 unpacker/universal_unpacker.py 目标文件路径 -o 输出目录 - 执行脱壳
技术堆栈
核心技术：
  • Python 3 - 核心分析引擎
  • Frida - 动态检测和注入框架
  • YARA - 恶意软件检测规则引擎
  • PE文件分析 - pefile库
  • Flask - Web界面后端
  • Bootstrap 5 - 前端UI框架
可选扩展：
  • Radare2 - 反汇编和分析框架
  • Capstone - 多平台反汇编引擎
  • UPX - 可执行文件压缩/解压工具
  • de4dot - .NET反混淆工具
  • Scylla - IAT重建工具
